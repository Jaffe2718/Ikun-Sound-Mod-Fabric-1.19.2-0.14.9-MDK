## 模组特色

> 本模组仅对原版游戏中村民和鸡受击时的音效以及玩家食用鸡肉的音效进行了爆梗式的更改，对于其他元素则保留了游戏的原汁原味，更多玩法和整活操作需要玩家自行探索。
